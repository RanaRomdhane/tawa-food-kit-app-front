```bash

```

- Install "Expo Go" app on your phone

- Scan the QR code that appears after npm start

- App opens automatically!

### Option B: Web Browser

```bash
npm run web
```

### Option C: Simulator

```bash
npm run android    # Android
npm run ios        # iOS (Mac only)
```
